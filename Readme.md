## 简介

### CoolRAR

是一款操作习惯和外观与WinRAR完全一致的软件。由于WinRAR个人版仅仅授权给个人，所以其不可以被商用。而很多人又不喜欢7zip一类软件的操作习惯，故产生了这款软件。

虽然这个软件本质上就是7zip换皮，但是使用起来还是相当舒服的。然而这个软件于2015年就已经停更，其解压内核已经跟不上现在的压缩包格式。御坂初琴软件屋这个版本基于CoolRAR1.7.3制作，**将其替换为19版7z内核，兼容RAR5格式**。

### QuickRAR

国产解压缩软件与WinRAR相比有一个最大的区别，就是国产解压缩软件打开文件时会锁定该文件，但WinRAR不会。好在我找到了一款不会锁定被打开的文件的解压缩软件：强压

虽然这个软件本质上就是7zip换皮，但是使用起来还是相当舒服的。然而这个软件于2015年就已经停更，其解压内核已经跟不上现在的压缩包格式。御坂初琴软件屋这个版本基于酷压V1.0制作，**将其替换为19版7z内核，兼容RAR5格式**。
